Live Night Heat
Illinois ECOSTRESS LST (70m) + data center overlays

Step-by-step guide (plain English)

  1. Background map (temperature): the app shows satellite-derived night surface temperature over Illinois. High-res ECOSTRESS is used when available; otherwise it falls back to a global thermal proxy.
  2. Data center points: each point is a known data center location from your Chicago list.
  3. AOI risk overlay: each polygon buffer has a risk score from recent heat behavior (anomaly, hot-night frequency, and trend).
  4. DC effect overlay (DC Δ vs controls): compares each data-center buffer temperature against matched non-data-center control buffers at the same timestamps.
  5. Before vs after opening: where opening-year metadata exists, popup stats show pre-open and post-open windows to help interpret possible change after go-live.
  6. Why coverage can look one-sided: ECOSTRESS is swath-based. A single scene may only cover part of Illinois; full-state appearance improves when latest same-timestamp tiles are mosaicked.
